# Manual de Usuario - Sistema PGD

## Introducción
El Sistema de Gestión de Escenarios Deportivos (PGD) permite administrar reservas, horarios y personal del Instituto.

## 1. Acceso al Sistema

### Iniciar Sesión
1. Abra su navegador web y diríjase a la dirección del sistema.
2. Ingrese su correo electrónico y contraseña.
3. Haga clic en "Ingresar".

*Nota: Si olvidó su contraseña, use la opción "¿Olvidaste tu contraseña?" para recibir un enlace de recuperación.*

## 2. Panel Principal (Dashboard)
Una vez autenticado, verá el Panel de Control con las siguientes opciones:
- Gestión de Reservas: Ver y aprobar solicitudes de espacios.
- Horarios: Consultar la disponibilidad de los escenarios.
- Personal: Ver la malla de turnos de los gestores.

## 3. Guía Paso a Paso

### Cómo Crear una Reserva
1. Navegue a la sección "Reservas".
2. Seleccione el Escenario Deportivo.
3. Elija la fecha y hora de inicio/fin.
4. Ingrese los datos del solicitante (Nombre y Teléfono).
5. Haga clic en "Guardar Reserva".
   - Verde: Aprobada.
   - Naranja: Pendiente de confirmación.

### Cómo Consultar Horarios de Personal
1. Vaya a "Subgerencia de Escenarios" > "Horario Gestores".
2. Busque por nombre de escenario o gestor.
3. Podrá ver los turnos de Lunes a Domingo y el contacto del gestor.

## 4. Soporte
Si encuentra algún error técnico, por favor contacte al administrador del sistema o envíe un reporte a `soporte@instituto.gov.co` indicando:
- Qué estaba haciendo cuando ocurrió el error.
- Captura de pantalla del mensaje de error (si aplica).
